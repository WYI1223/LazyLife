# Contributing

Project contribution guide lives at:

- `docs/index.md` (canonical docs entrypoint)
- `docs/governance/CONTRIBUTING.md`
- `docs/architecture/engineering-standards.md`
- `docs/architecture/code-comment-standards.md`
- `SECURITY.md` (for private vulnerability reporting)

Please read these documents before opening a PR.

## Docs Language Policy

- English docs are canonical.
- Translations are optional and may lag behind canonical docs.
- Internal links should point to canonical paths under `docs/`.
