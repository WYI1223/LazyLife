# Code of Conduct

## Purpose

LazyNote 社区致力于构建一个开放、尊重、可协作的技术环境。  
本行为准则适用于项目相关的所有互动场景：Issue、PR、讨论区、文档评论、线上会议与即时沟通。

## Expected Behavior

我们期望所有参与者：

- 尊重不同背景、经验和观点。
- 以事实和技术为中心讨论问题。
- 接受并提供建设性反馈。
- 在沟通中保持克制，避免人身攻击。
- 对失误负责，及时修正并总结。

## Unacceptable Behavior

以下行为不可接受：

- 侮辱、骚扰、歧视、威胁、恶意挑衅。
- 人身攻击、羞辱、持续性恶意争执。
- 发布隐私信息（如真实身份、联系方式、住址）或诱导他人泄露隐私。
- 明知故犯地破坏协作流程（刷屏、灌水、恶意提交）。
- 任何违法、恶意或明显损害社区安全的行为。

## Scope

本准则适用于：

- 仓库内所有公开互动（代码、文档、评论、讨论）。
- 以项目名义进行的外部活动与沟通。

## Reporting

如遇违规行为，请尽快联系维护者（maintainers）并提供：

- 事件时间与位置（链接、截图、上下文）。
- 涉及账号与行为描述。
- 你期望的处理方式（如删除内容、调解、封禁等）。

报告应被认真对待，维护者会尽可能保护举报人隐私。

## Enforcement

维护者有权根据情节采取措施，包括但不限于：

1. 口头或书面提醒。
2. 临时限制互动（锁帖、限制评论、限制提交）。
3. 撤销协作权限（移除协作者、封禁账号）。

严重或重复违规可直接进入更高等级处置。

## Appeals

被处置方可向维护者提出复核请求，并补充事实材料。  
维护者将基于证据进行复审，复审结果为最终决定。

## Maintainer Responsibility

维护者应：

- 公平、及时处理举报。
- 在保护隐私前提下记录处理结论。
- 以社区安全与协作效率为优先。
